## Pré-condições
- Usuário cadastrado no sistema
- Sistema acessível em cutaxis-pro.vercel.app

## Passos
1. Acessar a página de login do CutAxis Pro
2. Inserir e-mail válido no campo "Email"
3. Inserir senha correta no campo "Senha"
4. Clicar no botão "Entrar"

## Resultado Esperado
Usuário redirecionado para o dashboard com 
menu lateral e dados de estoque visíveis.
